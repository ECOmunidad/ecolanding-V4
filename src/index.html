Campos de Acción

Desarrollo de Software

Para aplicativos web y dispositivos móviles (IOS, Android, FirefoxOS, Windows Phone), desarrollo de software especializado en temas ambientales

Desarrollo de Hardware

Dispositivos electrónicos y/o mecánicos para mediciones ambientales, captura de datos, conectividad y eficiencia energética

Investigación

Análisis de información mediante procesos de Big Data, para obtención de datos, informes y estadísticas ambientales

Green Tic

Proyectos y consultoría en el área de Green Tic, para optimizar y hacer sostenibles las tecnologías de información y comunicación para empresas y comunidades
